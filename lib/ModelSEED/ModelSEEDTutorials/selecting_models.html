Selecting a Model

Filter Select    Model Table    User Models

Filter Select

There are multiple ways to select a model you are interested in.

By default, when the Model SEED page loads, the 'Selected models/FBA' tab will be selected. Here you can see the total number of models in the database, as well as the number of compounds and reactions.

A list of the available models can be found in the 'Select Model' tab. You can scroll through the list to browse the models, using the checkboxes to narrow the choices. If you find a model you want to examine, highlight the model in the list and click the 'Select Model' button.

Or if you already know the model, begin typing the organism name or id into the box and the list will be filtered.

Model Table

Select the 'Model statistics/Select' tab to see a table of the available models along with some additional information for each model. In this table, the columns are sortable and/or searchable to make finding a model that meets certain criteria easy. Click on the model link in the 'Genome ID' column to be taken to the organism page within the SEED (this will take you away from the Model SEED), or click on the model link in the 'Name' column to select the model for further comparison within the Model SEED page.

top

User Models

If you are signed in as a user, you can select personal models within the 'User models' tab. Clicking on the model link in the 'Name' column will select that model for examination and comparison.
